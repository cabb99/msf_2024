Getting Started
===============

Installation
------------

In order to install this program you have to execute the following command

```
pip install -e .
```

Tutorial
--------
Some common things that you can do are

### Import xyz

```
import msf_2024
```

### Open xyz

```
msf_2024.io.open_xyz
```

### Open pdbs

```
msf_2024.io.open_pdb
```

### Calculate distances

```
msf_2024.calculate_distances
```
